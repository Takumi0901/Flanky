Flanky

自作CSSフレームワーク
Sass + Flocss でFlexレイアウト、レスポンシブ対応のフレームワークです。

ゴーストボタン

概要

対応ブラウザについて

  • Android OS 4.1以上

  • iOS 6以上

  • IE10以上

  • Chrome最新

  • FireFox最新

  • Safari最新

Sassファイル(CSSファイル)のディレクトリ構成

ディレクトリ構成は、FLOCSSを参考にしています。

sass
  ├─ foundation
  |     ├─ mixins
  |     ├─ _base.scss
  |     ├─ _mixins.scss
  |     ├─ _sanitize.scss
  |     └─ _variables.scss
  ├─ layout
  |     ├─ _header.scss
  |     └─ _footer.scss
  ├─ object
  |     ├─ componet
  |     ├─ project
  |     └─ utility
  └─ app.scss

※object/projectについては、使用するサイトに合わせたスタイルを書くのがいいと思います。

接頭辞について

なんの属性なのかHTMLのコードを見ただけですぐにわかるように接頭辞をつけることを義務付け

layoutを表す接頭辞

.l-

componetを表す接頭辞

.c-

projectを表す接頭辞

.p-

utilityを表す接頭辞

.u-

状態を表す接頭辞

.is-

例えば、is-active、is-hiddenなど。

JavaScriptを表す接頭辞

.js-

cssのスタイルが定義されたセレクタとJavaScriptで動的に動かすセレクタを混同させないこと。
